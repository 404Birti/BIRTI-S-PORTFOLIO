Welcome to CSS

CSS (Cascading Style Sheets) is a language used to style and design web pages. It allows you to change colors, fonts, spacing, backgrounds, borders, and the overall layout of a webpage. CSS makes websites more attractive, organized, and user-friendly.

Why is CSS Important?

CSS separates content (HTML) from presentation (styles), making websites easier to maintain and reuse across multiple pages.
CSS is mainly used for:

• Styling – Colors, fonts, sizes, borders, and backgrounds.
• Layout – Flexbox, Grid, positioning, and spacing.
• Responsiveness – Adapting webpages to different screen sizes.
• Reusability – One stylesheet can be used for many webpages.
CSS Syntax

Basic syntax:

selector {
  property: value;
}

Example:

body {
  background-color: lightblue;
  font-family: Arial, sans-serif;
}
Here, body is the selector, while background-color and font-family are properties with their respective values.
Types of CSS

1. Inline CSS
Applied directly to an HTML element using the style attribute.

Example:
<p style="color:red; font-size:20px;">Text</p>

Pros: Immediate control over a single element.
Cons: Difficult to maintain and cannot be reused.

2. Internal CSS
Defined inside the HTML file using the <style> tag.

Example:
<style>
  h1 { color: blue; }
</style>

Pros: Easy to manage styles for one webpage.
Cons: Cannot be reused across multiple pages.

3. External CSS
Stored in a separate .css file and linked to the HTML page.

Example:
<link rel="stylesheet" href="style.css">

Pros: Reusable and easy to update.
Cons: Requires an additional CSS file.
Key Features of CSS

• Cascading – Resolves style conflicts automatically.
• Selectors – Target specific HTML elements.
• Media Queries – Create responsive webpages for different devices.
• Reusability – One stylesheet can style multiple pages.
Conclusion

CSS is an essential technology for modern web development. It improves the appearance of webpages, makes them responsive, and allows developers to maintain consistent designs across an entire website.
