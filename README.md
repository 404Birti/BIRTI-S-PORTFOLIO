# Personal Portfolio Website

## About the Project

This is a personal portfolio website created as part of my Web Development coursework. The website is built using HTML and CSS and consists of multiple webpages connected through a navigation bar. It showcases my personal information, projects, and contact details while demonstrating the HTML and CSS concepts I have learned.

## Features

- Home page with introduction
- About Me page
- Projects page
- Contact page
- Internal CSS styling
- Navigation between pages
- Images and buttons
- Tables and forms
- Responsive and simple layout
